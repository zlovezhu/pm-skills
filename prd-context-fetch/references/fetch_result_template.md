# 参考清单输出模板

> 给 `prd-context-fetch` Skill 用。检索完成后用这份模板回给用户勾选。

## 一、模板原文（标准情况，wiki / card 有命中）

```
🔍 PRD 参考清单 · {目标标题或模块名}
─────────────────────────
检索范围：wiki/ + card/ + TAPD 当前项目同模块（近 6 个月）
匹配模块：{选中的 tag}

📚 Wiki 命中（{N} 篇）
[x] {标题}
    依据：{一句话相关度判断}
    🔗 {Obsidian 链接}
[x] {标题}
    依据：...
[ ] {标题}（弱相关，默认不选）
    依据：...

🃏 Card 命中（{N} 张）
[x] {标题} — {card 全文一句话}
[ ] {标题} — ...

📋 TAPD 同模块历史（{N} 条）
[x] [{TAPD ID}] {标题} · {状态} · {修改时间}
    关键点：
      · {边界/约束/决策点 1}
      · {边界/约束/决策点 2}
[ ] [{TAPD ID}] {标题} · ...

─────────────────────────
请勾选要保留的项（默认勾选已经标 [x]，回我"用默认"或"全要"或"只要 wiki 第 1、2 条"即可）。
```

## 二、模板原文（兜底降级，wiki + card 都为空）

```
⚠️ 当前模块在 wiki/ 和 card/ 没有命中任何沉淀，已降级到 raw/ 检索。
raw 是未消化的原始素材，仅供参考，不应直接作为 PRD 依据。
建议本次写完 PRD 后用 `case-distill` 把这块沉淀成 wiki。

🔍 PRD 参考清单 · {目标标题或模块名}
─────────────────────────
检索范围：raw/ + TAPD 当前项目同模块（近 6 个月）
匹配模块：{选中的 tag}

📁 Raw 兜底（{N} 篇，仅供参考）
[ ] {标题}
    依据：{一句话相关度判断}
    🔗 {Obsidian 链接}
[ ] {标题}
    ...

📋 TAPD 同模块历史（{N} 条）
[x] [{TAPD ID}] {标题} · {状态} · {修改时间}
    关键点：
      · ...

─────────────────────────
请勾选要保留的项（raw 默认全部不勾，需要你显式选）。
```

## 三、完整示例

输入：「想做{某模块} {某模式}下的取消操作」

输出：

```
🔍 PRD 参考清单 · {模块} {模式} 取消操作
─────────────────────────
检索范围：wiki/ + card/ + TAPD 当前项目同模块（近 6 个月）
匹配模块：#{模块}（强匹配）  #{模式}（强匹配）

📚 Wiki 命中（2 篇）
[x] {模块} {模式} 对比
    依据：标题命中"{模块} + {模式}"，正文涉及操作交互实现
    🔗 obsidian://Wiki/{模块}%20{模式}%20对比.md
[x] {另一模块} 性能优化案例
    依据：提及"操作交互"的性能边界，可能影响取消按钮响应
    🔗 obsidian://1-Projects/<YOUR_PROJECT>/{另一模块}性能优化案例.md

🃏 Card 命中（1 张）
[x] B 端专业工具的核心价值是缩短从发现到定位的路径
    — 这条是原则提示：取消操作属于"快速回到初始视图"的链路，按此原则应保证 < 200ms 响应

📋 TAPD 同模块历史（3 条）
[x] [1234561] {模块} 支持{某交互} · 已发布 · 2026-04-12
    关键点：
      · 操作范围用 [start, end] 时间戳表示
      · 跨子图切换时清空选区
      · 未做"取消按钮"，本次需求是补这个缺口
[x] [1234555] {模块} 缩放与拖拽 · 已发布 · 2026-03-08
    关键点：
      · 缩放/拖拽时不触发重算
      · 取消缩放走双击空白区
[ ] [1234540] {模块} 颜色配色方案 · 已发布 · 2026-02-21
    （弱相关，默认不选）

─────────────────────────
请勾选要保留的项（默认勾选已经标 [x]，回我"用默认"或"全要"或"只要 wiki 第 1 条"即可）。
```

## 四、确认后的输出

用户回了"用默认"或者"只要 wiki 第 1 条"之后，Skill 输出一段确认：

```
✅ 已锁定 N 条参考材料，组装成 context 包：
  · Wiki: {模块} {模式} 对比、{另一模块} 性能优化案例
  · Card: 缩短发现到定位路径
  · TAPD: 1234561、1234555

下一步可以直接接 prd-writer 开始写 PRD。是否启动？
```
